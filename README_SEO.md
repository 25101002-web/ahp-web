On-Page & Off-Page SEO checklist for this static site

On-Page
- Add unique, descriptive <title> and <meta name="description"> (done in `index.html`).
- Use structured data (JSON-LD) for Organization and WebSite (done).
- Add Open Graph and Twitter Card metadata for share previews (done).
- Ensure images have `alt` attributes and meaningful filenames.
- Ensure canonical tags point to production URL.
- Create `sitemap.xml` and `robots.txt` (done).

Off-Page
- Submit sitemap to Google Search Console and Bing Webmaster Tools.
- Add backlinks from authoritative pages (university site, social profiles).
- Share the site on relevant forums and social channels.
- Consider SSL (HTTPS) via GitHub Pages or your hosting provider.

Performance
- Host images with optimized sizes (compress Nha images).
- Use a CDN for static assets if possible.

Privacy & Legal
- If collecting user info (report form), ensure privacy policy and secure storage.
